## Modelo de Dados (dentro do index.html)

```js
const OWNER = { name: "Renato", whatsapp: "5519996614321" }; // 55 + DDD 19 + número
const ITEMS = [ { id, title, description, pricePaid, askingPrice, refs[], photos[], sold }, ... ];
```

- `pricePaid` = preço "de" (quanto custou). `askingPrice` = preço "por" (quanto quer).
- `sold: true` → item fica cinza, tachado e sem botão de lance.
- "dar um lance" → abre `wa.me` do OWNER com mensagem pré-preenchida.
